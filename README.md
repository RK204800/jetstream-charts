# Jetstream Charts

Static Next.js site rendering Stage 9b visual trade audits for Jetstream strategies.

Live: https://jetstream-charts.vercel.app

## Pages

- `/` — index of strategies with charts available
- `/<slug>/` — per-strategy view: candlestick chart, last 10 trades with entry/exit markers, summary panel, clickable trade list

## Updating

Per-strategy data lives in `public/data/<slug>.json`. Generated by
`jetstream-hmm/nautilus/visualizer/build_chart.py`. Push to main triggers auto-redeploy on Vercel.

## Stack

- Next.js 15 App Router with `output: 'export'` (static)
- TradingView Lightweight Charts v5 (uses `createSeriesMarkers` from main package)
- Tailwind CSS
- Hosted on Vercel free tier

## Schema

Per-strategy JSON conforms to `Business-Units/Jetstream/Strategies/_schema/09b-trade-view.schema.json` in the obsidian-vault repo.
